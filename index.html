Animate It!

Built with a Powerful Animation Framework

  • PwrSlide.html ships with over 50 animations from Animate.css
  • PwrSlides and PwrPieces descend from the PwrAnimated Object
    • Custom attributes give fine-grain control over each animated component of your presentation
      •
    • Animated components have access to several powerful animation methods
      •
  • Wrap any regular HTML element in <pwr-piece> tags for them to become animated
    • Tags are animated by the slide in descending order
      •
    • Add the 'show' attribute to the component to show its contents initially
      •

PwrAnimated


<pwr-animated entrance exit onentry onexit>
Attributes
  • entrance: specify the entrance animation name
  • exit: specify the exit animation name
  • onentry: optional callback executed after entrance animation
  • onexit: optional callback executed before exit animation
    • NOTE: callbacks are bound by default to the tag on which they are called

PwrAnimated


PwrAnimated = {
	animateIn: function() { ... },
	animateOut: function() { ... },
	animateEl: function(animationName) { ... },
	removeAnimations: function() { ... }
};
Methods
  • animateIn( ): queue the target component's entrance animation
  • animateOut( ): queue the target component's exit animation
  • animateEl(animationName): applies the given animationName to the component
  • removeAnimations( ): removes any animations from the component
